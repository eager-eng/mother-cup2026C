# 中老年人群高血脂症风险预警及干预方案优化

本项目从原始附件直接复现三个问题。全部数据处理、模型求解、制图、结果导出和核心验收逻辑均位于一个 Python 文件中，并按问题一、问题二、问题三顺序编排，便于复制到论文附录。

## 运行

```powershell
py -3.12 -m venv .venv312
.\.venv312\Scripts\python.exe -m pip install -r requirements-py312.txt
.\.venv312\Scripts\python.exe code\中老年人群高血脂症建模完整代码.py
```

完整运行约需 1—2 分钟。运行结束会自动执行问题一至三的关键口径与约束检查，并生成：

- `reports/RESULTS_REPORT.md`
- `results/*.csv|json`
- `results/预处理结果/高血脂症数据_预处理完成.csv`
- `results/预处理结果/高血脂症数据_预处理完成.xlsx`
- `figures/*.pdf`（论文与 LaTeX 插图的权威矢量版本）
- `figures/png/*.png`（与 PDF 同名的 300 dpi 预览版本）

预处理结果没有丢失：旧版重复 CSV/XLSX/PNG 曾按目录整理方案主动删除；当前代码现已从原始附件重新生成最终模型口径的数据文件，并新增核心指标相关性热力图和标准化箱线图。项目根目录不再存放重复图片。

`code/` 中仅有 `中老年人群高血脂症建模完整代码.py`，其中六个带编号的分区可直接定位公共预处理、三个问题、结果验收和主入口。

## 建模边界

附件的高血脂标签与四项血脂任一异常规则完全一致。四项血脂不会进入高血脂关联筛查模型；非血脂概率只解释为与当前异常的横断面关联分数。最终低、中、高三级统计等级由训练集折外概率推导，浅层 CART 提供特征阈值解释；题面示例规则单独保留为临床管理提示。附件没有随访结局，项目不宣称能够预测未来发病。

问题三先按题面痰湿积分、年龄和活动能力阈值形成9类确定性患者分层，再以受限多输出CART解释动态规划的首月活动强度与频率。CART是快速初筛代理，完整六个月处方始终以硬约束和动态规划结果为准。
