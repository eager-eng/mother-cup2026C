# 建模报告

根据题目动态调整问题数量。本赛题共 3 个子问题。

## 1. 总体建模框架

题面与附件 → 原始数据一致性校验 → 问题一双终点筛选与体质贡献 → 问题二非血脂未病预警与临床覆盖规则 → 问题三个体化动态规划 → 阈值与参数敏感性 → 约束回代。

## 2. 子问题拆解

### 问题一

- 输入：血脂、代谢指标、活动量表、九体质积分、高血脂标签。
- 输出：同时表征痰湿严重程度和高血脂关联的稳定指标；九体质调整 OR。
- 评价：Bootstrap 稳定频率、双终点共识得分、Spearman 相关、单指标 AUC、OR 的 95%CI/P 值/VIF。
- 图表：共识得分、双终点频率、九体质 OR 置信区间。

### 问题二

- 输入：九体质、活动总分、血糖、尿酸、BMI、年龄、性别、烟酒史。
- 预测对象：高血脂二分类标签。
- 输出：低、中、高三级未病预警；概率阈值依据；痰湿质高危组合。
- 评价：训练集交叉验证 AUC、独立测试 AUC/AP/Brier、三级人数与实际患病率。
- 图表：ROC、三级分布与实际患病率、校准曲线。

### 问题三

- 输入：体质标签 5 患者的样本 ID、痰湿积分、年龄组、活动总分及题面成本/疗效参数。
- 决策变量：每月调理等级、活动强度、每周频率。
- 目标：在成本不超过 2000 元时，先得到最大可降低分数，再选择达到其 90% 以上且成本最低的方案。
- 约束：年龄、活动评分、强度、频率、6 个月周期、预算。
- 输出：全部痰湿质患者推荐；ID 1/2/3 逐月方案；特征—方案匹配规律；Pareto 与灵敏度。

## 3. 假设敏感性预检

### 模糊表述及解释

1. “预警”可解释为用当前四项血脂复现诊断，也可解释为不依赖诊断血脂的早期风险排序。附件标签由四项血脂异常规则完全决定，前者会产生诊断同义反推。
2. 题面给出中医调理方式与成本，但未给其独立积分下降率；可自行赋值，也可在基准模型中仅计成本。
3. “考虑经济成本及有效降低”可用加权和，也可用 ε-约束/Pareto。

### 快速验算与递进性检查

- 四项血脂任一异常与标签一致率为 100%，故将其 AUC 作为未病预警证据会夸大能力。
- 低于每周 5 次时题面说明积分基本稳定；状态转移不得虚构下降。
- 直接最小化最终积分会倾向耗尽预算，不能体现经济性。

### 最终采用的解释

- 未病预警模型排除当前四项诊断血脂；若已出现血脂异常，则临床覆盖规则直接进入高危管理。
- 基准模型不编造中医调理独立疗效，只计题面给定成本；额外月降幅 0%/1%/2% 作为敏感性场景。
- 问题三用 ε-约束：达到预算内最大效果的 90% 后最小化成本。

### 绘制的图像和对比表格

- 问题一：稳定筛选条形图、频率热力图、OR 置信区间图。
- 问题二：ROC、风险分层、校准曲线、阈值 Bootstrap 表、组合规则表。
- 问题三：ID 1/2/3 积分曲线、成本效果对比、匹配规律表、敏感性表。

## 4. 数据处理方案

- 原始附件 1000 行、37 列；样本 ID 必须唯一。
- ADL+IADL 必须与活动总分恒等。
- 临床阈值：TC>6.2、TG>1.7、LDL-C>3.1、HDL-C<1.04 判异常。
- 不同时把 ADL、IADL 与其总分放入同一模型，避免完全线性重复。
- 预处理过程由 `code/solution.py` 从原附件重建，不依赖旧中间文件。

## 5. 问题一模型

每轮 Bootstrap 同时训练随机森林回归（痰湿积分）和分类（高血脂标签），并把真实特征重要性与逐列置乱影子特征最大重要性比较。定义稳定频率为真实重要性超过影子最大值的比例。

双终点共识分数为：

\[
C_j=\sqrt{f_{j,S}f_{j,Y}}\sqrt{I_{j,S}^{*}I_{j,Y}^{*}},
\]

其中两终点重要性分别归一化。入选需两个稳定频率均不低于 0.60，且共识得分位于前 30%。

九体质贡献采用无惩罚多变量 Logistic，连续变量标准化。调整变量为年龄、性别、烟酒、BMI、活动总分、血糖和血尿酸。由极大似然 Hessian 估计系数标准误与 95%CI。

## 6. 问题二模型

候选模型为加权 Logistic 与随机森林，只在训练集五折交叉验证比较 AUC。胜出模型用五折 Platt 校准。低中阈值与中高阈值只在训练集 OOF 概率上搜索，约束每层占比不低于 15%，优先使实际患病率单调并扩大高低层差异。

痰湿质核心组合按“体质标签=5 + 临床阈值条件”枚举一至两个附加条件，报告支持度、置信度和提升度。

## 7. 问题三模型

状态为月末痰湿积分与累计成本；动作是活动强度和每周频率。每月调理等级由月初积分唯一确定。活动下降率为：

\[
r(a,f)=\begin{cases}
0,&f<5,\\
0.03(a-1)+0.01(f-5),&f\ge5.
\end{cases}
\]

状态转移为 \(S_{t+1}=S_t(1-r)\)。同一离散积分仅保留最低成本，并删除成本和积分均被支配的状态。最终先求最大降幅 \(\Delta_{max}\)，再最小化满足 \(\Delta\ge0.9\Delta_{max}\) 的成本。

## 8. 灵敏度分析与检验方案

- 问题二：对训练集 OOF 结果 Bootstrap 100 次，观察两个阈值稳定性。
- 问题三：分别扰动积分步长、活动效果系数、预算上限和调理额外月降幅。
- 每位患者回代频率、强度、年龄/评分耐受度、预算、月数与积分单调约束。
