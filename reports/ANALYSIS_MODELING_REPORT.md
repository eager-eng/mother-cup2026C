# 建模报告

根据题目动态调整问题数量。本赛题共 3 个子问题。

## 1. 总体建模框架

题面与附件 → 原始数据一致性校验 → 问题一分终点折外筛选、探索性 Pareto 与体质排名稳定性 → 问题二非血脂关联模型、训练内概率分层与 CART 解释 → 问题三个体化动态规划和 Pareto 选点 → 参数敏感性 → 约束回代。

## 2. 子问题拆解

### 问题一

- 输入：血脂、代谢指标、活动量表、体质标签与九体质积分、高血脂标签。
- 输出：痰湿严重度指标集、排除诊断泄漏后的高血脂关联指标集、两集合的真实交集、非血脂探索性 Pareto 候选；九种体质标签相对平和质的调整 OR 与 Bootstrap 排名稳定性。
- 评价：重复五折折外 R²/AUC、折外置换重要性及正重要性频率、Spearman/Mann-Whitney 检验、FDR、OR 的 95%CI、整体似然比检验和500次分层Bootstrap名次。
- 图表：分终点折外置换重要性、正重要性频率、分类体质 OR 置信区间。

### 问题二

- 输入：九体质、活动总分、血糖、尿酸、BMI、年龄、性别、烟酒史。
- 预测对象：高血脂二分类标签。
- 输出：当前诊断状态、非血脂关联筛查概率、训练集折外概率推导的唯一低/中/高统计等级、CART代理路径、独立临床管理提示和痰湿质高危组合。
- 评价：训练集交叉验证 AUC、独立测试 AUC/AP/Brier、三级人数与当前异常率、相邻等级 Fisher 检验、CART宏平均F1与层级一致率。
- 图表：ROC、三级分布与实际患病率、校准曲线。

### 问题三

- 输入：体质标签 5 患者的样本 ID、痰湿积分、年龄组、活动总分及题面成本/疗效参数。
- 决策变量：每月调理等级、活动强度、每周频率。
- 目标：在成本不超过 2000 元时，先得到最大可降低分数，再选择达到其 90% 以上且成本最低的方案。
- 约束：年龄、活动评分、强度、频率、6 个月周期、预算。
- 输出：全部痰湿质患者推荐；ID 1/2/3 逐月方案；特征—方案匹配规律；Pareto 与灵敏度。

## 3. 假设敏感性预检

### 模糊表述及解释

1. “预警”可解释为用当前四项血脂复现诊断，也可解释为不依赖诊断血脂的关联筛查。附件标签由四项血脂异常规则完全决定，且没有随访结局，因此既要防止诊断同义反推，也不能把横断面概率解释成未来发病概率。
2. 问题一要求指标同时满足两个终点，但附件可能不支持非空交集；强制选出共同指标会制造假阳性。
3. 题面给出中医调理方式与成本，但未给其独立积分下降率；可自行赋值，也可在基准模型中仅计成本。
4. “考虑经济成本及有效降低”可用加权和，也可用 ε-约束/Pareto。

### 快速验算与递进性检查

- 四项血脂任一异常与标签一致率为 100%，故将其用于高血脂终点筛选会产生标签泄漏。
- 旧方案中多数指标的训练内痰湿入选频率接近 1，但五折折外 R² 为负，说明必须改用折外证据并允许空集合。
- 低于每周 5 次时题面说明积分基本稳定；状态转移不得虚构下降。
- 直接最小化最终积分会倾向耗尽预算，不能体现经济性。

### 最终采用的解释

- 问题一拆分两个终点：痰湿端可考察全部指标，高血脂端排除四项诊断血脂及其派生量；共同指标只取两个通过集合的交集，允许为空。
- 九体质贡献以体质标签为分类暴露、平和质为参照；九个连续积分分别调整作为敏感性分析。允许报告调整OR点估计排名及Bootstrap稳定性，但必须与整体检验、置信区间和FDR共同解释。
- 问题二保留非血脂随机森林作为当前异常的关联筛查分数；三级统计等级由训练集折外概率推导，浅层CART负责给出可读的特征切点。题面示例阈值仅生成独立临床管理提示。
- 基准模型不编造中医调理独立疗效，只计题面给定成本；额外月降幅 0%/1%/2% 作为敏感性场景。
- 问题三用 ε-约束：达到预算内最大效果的 90% 后最小化成本。

### 绘制的图像和对比表格

- 问题一：折外置换重要性条形图、正重要性频率热力图、分类体质 OR 置信区间图。
- 问题二：非血脂 ROC、统计三级分层、校准曲线、训练内阈值表、CART规则表、组合规则表。
- 问题三：ID 1/2/3 积分曲线、成本效果对比、成本—降幅 Pareto 前沿、匹配规律表、敏感性表。

## 4. 数据处理方案

- 原始附件 1000 行、37 列；样本 ID 必须唯一。
- ADL+IADL 必须与活动总分恒等。
- 临床阈值：TC>6.2、TG>1.7、LDL-C>3.1、HDL-C<1.04 判异常。
- 不同时把 ADL、IADL 与其总分放入同一模型，避免完全线性重复。
- 预处理过程由统一建模程序从原附件重建，不依赖旧中间文件。

## 5. 问题一模型

痰湿严重度终点使用 TC、TG、LDL-C、HDL-C、血糖、血尿酸、BMI 和活动总分。采用 5 折、3 次重复交叉验证；每一折只在训练子集拟合随机森林回归，在验证折计算 R² 和置换重要性。指标入选必须同时满足：整体平均折外 R²>0、平均置换重要性>0、正重要性频率≥0.60、|Spearman ρ|≥0.10 且 FDR q<0.05。

高血脂关联终点排除 TC、TG、LDL-C、HDL-C 及其异常派生量，仅使用血糖、血尿酸、BMI 和活动总分。采用重复分层五折随机森林，记录折外 AUC 和验证折置换重要性。指标入选要求平均折外 AUC≥0.60、正重要性频率≥0.60、单指标 AUC≥0.60 且 Mann-Whitney 检验 FDR q<0.05。最终共同指标定义为两个入选集合的交集，不设非空约束。

为增强阴性结果的可解释性，对四个非诊断候选指标分别取两个终点的正向折外置换重要性并归一化；若不存在另一指标在两维均不差且至少一维更优，则标记为探索性 Pareto 候选。该标记只用于展示多目标权衡，不改变验证性门槛。

九体质主分析以体质标签为分类暴露，以平和质为参照，调整年龄、性别、烟酒、BMI、活动总分、血糖和血尿酸：

\[
\operatorname{logit}P(Y=1)=\beta_0+\sum_{k=2}^{9}\beta_k I(T=k)+\sum_m\gamma_mX_m.
\]

先用全模型与仅含协变量模型的似然比检验判断九种体质整体差异，再报告各标签 OR、95%CI、P 值和 BH-FDR。八种偏颇体质按调整OR给出探索性点估计名次，并在结局分层重抽样的500次Bootstrap中报告中位名次、95%名次区间和首位频率。九个连续体质积分分别进入调整模型作为敏感性分析；排名不替代整体检验，也不作因果解释。

## 6. 问题二模型

候选非血脂关联模型为加权 Logistic、随机森林与浅层 CART，只在训练集五折交叉验证比较 AUC，胜出模型用五折 Platt 校准。模型特征限定为九体质积分、活动总分、血糖、血尿酸、BMI、年龄、性别和烟酒史，不包含四项诊断血脂及其异常派生量。模型概率只解释为对当前血脂异常的关联筛查分数。

训练集采用外层五折、内层三折校准生成折外概率。在排序后的折外概率上遍历两个切点，要求各层至少占训练样本15%、三层观察患病率单调不下降，并最大化加权层间患病率方差。切点确定后一次性应用于独立测试集，测试集不得参与阈值选择。

为把随机森林分数转化为特征级阈值，使用低、中、高统计等级作为代理标签，在最大深度2/3/4、最小叶节点比例5%/10%、最多6个叶节点的网格中，按训练交叉验证宏平均F1和一致率选择浅层CART。输出每个叶节点的实际特征阈值、代理等级、样本量和观察患病率。

每名患者输出诊断状态、非血脂筛查概率、唯一统计风险等级、CART代理路径以及题面示例规则对应的临床管理提示。痰湿质核心组合以统计高危层为结局，前件排除诊断血脂，枚举一至两个附加条件，报告覆盖人数、高危人数、支持度、置信度及 Jeffreys 95%CI、提升度。

## 7. 问题三模型

状态为月末痰湿积分与累计成本；动作是活动强度和每周频率。每月调理等级由月初积分唯一确定。活动下降率为：

\[
r(a,f)=\begin{cases}
0,&f<5,\\
0.03(a-1)+0.01(f-5),&f\ge5.
\end{cases}
\]

状态转移为 \(S_{t+1}=S_t(1-r)\)。同一离散积分仅保留最低成本，并删除成本和积分均被支配的状态。最终先求最大降幅 \(\Delta_{max}\)，再最小化满足 \(\Delta\ge0.9\Delta_{max}\) 的成本。

为把个体优化结果转化为可追溯的“患者特征—方案”规律，增加两层解释结构。第一层严格使用题面阈值，将痰湿积分的3个调理等级与年龄、活动能力共同确定的3个最大耐受强度交叉形成9类患者分层。最大耐受3级要求年龄组1—2且活动总分不低于60；最大耐受2级包括年龄组1—2且活动总分为40—59，或年龄组3—4且活动总分不低于40；其余患者归入最大耐受1级。

第二层以动态规划给出的首月活动强度和每周频率作为双输出，使用年龄组、活动总分和初始痰湿积分训练受限CART解释代理。最大深度候选为2—6，最小叶节点比例候选为5%、8%、10%，最大叶节点数候选为6、9、12；五折KFold按联合处方一致率、分量一致率、叶节点数和树深度依次选择，并排除产生耐受冲突的候选。CART仅用于快速解释首月决策，完整六个月方案以及任何冲突情形均以硬约束和动态规划为准。

## 8. 灵敏度分析与检验方案

- 问题二：记录训练集折外概率切点的目标值、各层样本量与患病率，并比较CART参数网格的宏平均F1和一致率。
- 问题三：分别扰动积分步长、活动效果系数、预算上限和调理额外月降幅；同时报告CART五折与训练内一致率，避免把代理树解释为新的疗效模型。
- 每位患者回代频率、强度、年龄/评分耐受度、预算、月数与积分单调约束。
